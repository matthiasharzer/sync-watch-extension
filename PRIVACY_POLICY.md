# Privacy Policy

**Effective Date:** March 26, 2026

This Privacy Policy applies to the **Sync Watch** Chrome Extension ("the Extension").

## 1. Data Transmission and Usage
To synchronize video playback with other users, the Extension transmits limited, anonymous operational data to our backend server. This data consists strictly of:
* A unique message ID
* A timestamp
* The current video playing state (e.g., play, pause)
* The current video playback time
* The room ID to identify which users to synchronize with

**No personal information, account details, browsing history, or sensitive data is ever collected or transmitted.**

## 2. Data Storage and Retention
Our backend server functions solely as a real-time message distributor to synchronize clients connected to the same room. **No messages, logs, or playback data are saved or stored on our servers.** Once a playback message is relayed to the room, it is immediately discarded. 

## 3. Open Source
The Extension is entirely open-source. You are welcome to review the source code to verify our data handling and privacy practices. The code for the Extension is available in this repository, and the backend server code is available at https://github.com/matthiasharzer/sync-watch-server.

## 4. Changes to this Policy
If we ever change our practices regarding data transmission or storage, this Privacy Policy will be updated accordingly.
